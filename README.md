# hackpad

This is my (shrey's) hackpad

Features:

Case EC11 Rotary encoder for sound and volume (please double check if i did it right) 2 WS2812B RGB LEDs. 4 Keys

CAD Model:

This has 2 parts to the case, which will be held together with 4 M3 Screws

Firmware:

This hackpad uses QMK firmware for everything.

The rotary encoder changes volume
The 4 keys currently act as macros

BOM:

4x Cherry MX Switches 
4x DSA Keycaps
4x M3x16mm SHCS Bolts 
2x WS2812B LEDs
1x EC11 Rotary Encoder
1x XIAO RP2040 
1x Case
